# ☕ Космический бариста — Coffee Platformer

Игра-платформер на JavaScript (Canvas API), созданная в рамках учебного задания.

## 🎮 Об игре

Космический бариста путешествует по галактике, собирая кофейные зёрна на парящих платформах и躲避ая капли пролитого кофе.

## 🕹️ Управление

- **← → / A D** — движение
- **Пробел / W** — прыжок
- **Esc** — пауза

## ✨ Функционал

- Процедурная генерация уровней
- Физика персонажа (гравитация, коллизии)
- Сбор кофейных зёрен (+100 очков)
- Враги с патрулированием
- Система жизней (3 ❤️)
- Система частиц
- Параллакс-фон со звёздами
- Сохранение рекорда в localStorage
- Адаптивный дизайн
